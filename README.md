# Task Manager - To-Do List App

A simple and interactive To-Do list application built with Vite + React.js where you can assign tasks to people and track their completion.

## Features

- ✨ Add tasks with person name and task description
- ✅ Check/uncheck tasks to mark completion
- 🗑️ Delete tasks
- 🎨 Clean and interactive UI
- 📱 Responsive design

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production
```bash
npm run build
```

## How to Use

1. Enter the person's name in the "Person Name" field
2. Enter the task description in the "Task Description" field
3. Click "Add Task" to add the task to the list
4. Click the checkbox to mark a task as completed
5. Click the × button to delete a task

Enjoy managing your tasks!
